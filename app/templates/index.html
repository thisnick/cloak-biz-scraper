How many browsers can run at once. Each running browser uses roughly 0.5–1 GB, billed on actual usage (about $10/GB per month). If you turned on Serverless in Railway, the server sleeps when idle and you pay only for the minutes a sweep runs — a 3-minute sweep with 4 browsers costs pennies. If you didn't, you're paying for every hour of the month (~$8–9) whether you scrape or not.
